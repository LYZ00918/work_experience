# 上传代码步骤
## 1.git init
    1. 先进入文件名project,再git init
    2. 在任意位置，然后加入文件地址： git init D:project
    如果有些文件不想要，可以增加一个.gitignore 文件
## 2.git add .
    添加文件
## 3.git commit -m '第一次任务提交'

## 4.git remote add origin https://github.com/LYZ00918/work_experience.git

## 5.git push -u origin master:main
